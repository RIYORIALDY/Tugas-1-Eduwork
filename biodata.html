Instruksi GitHub

Membuat Repositori GitHub

  1. Buka GitHub dan masuk ke akun Anda.
  2. Klik tombol "+" di sudut kanan atas dan pilih "New repository".
  3. Beri nama repositori dan tambahkan deskripsi jika diperlukan.
  4. Pilih opsi "Public" atau "Private" sesuai kebutuhan.
  5. Klik tombol "Create repository".

Menginisialisasi Repositori Lokal

$$git init$$
$$git add .$$
$$git commit -m "Initial commit"$$

Mendorong Kode ke GitHub

$$git remote add origin https://github.com/username/nama-repositori.git$$
$$git branch -M main$$
$$git push -u origin main$$
